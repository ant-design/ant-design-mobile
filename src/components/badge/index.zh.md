<code src="./demos/index.tsx"></code>

### 属性

| 属性    | 说明                                                                                            | 类型                                  | 默认值    |
| ------- | ----------------------------------------------------------------------------------------------- | ------------------------------------- | --------- |
| content | 徽标内容：如果传 `null` `undefined` `''` 或不传，则不显示徽标；如果传 `Badge.dot`，会显示小红点 | `React.ReactNode \| typeof Badge.dot` | -         |
| color   | 徽标背景色                                                                                      | `string`                              | `#FF411C` |

### CSS 变量

| 属性    | 说明                       | 默认值 |
| ------- | -------------------------- | ------ |
| --right | 相对于最右边，向左的偏移量 | `0`    |
| --top   | 相对于最上边，向下的偏移量 | `0`    |
