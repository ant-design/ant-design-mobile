# Divider 分割线

区隔内容的分割线。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

## Divider

### 属性

| 属性            | 说明                                              | 类型                            | 默认值         |
| --------------- | ------------------------------------------------- | ------------------------------- | -------------- |
| contentPosition | 内容位置，仅在 `direction` 为 `horizontal` 时有效 | `'center' \| 'left' \| 'right'` | `'center'`     |
| direction       | 水平还是垂直类型                                  | `'horizontal' \| 'vertical'`    | `'horizontal'` |
