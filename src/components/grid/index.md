# API

## Grid

| 属性    | 说明           | 类型                                             | 默认值 |
| ------- | -------------- | ------------------------------------------------ | ------ |
| columns | 列数           | number                                           | -      |
| gap     | 格子之间的间距 | `number` \| `number[]` \| `string` \| `string[]` | 0      |

## Grid.Item

| 属性 | 说明 | 类型   | 默认值 |
| ---- | ---- | ------ | ------ |
| span | 跨度 | number | 1      |
